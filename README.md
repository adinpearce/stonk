# stonk


整個專案做為追蹤及評估隔日沖券商分點並使用line機器人作為前端資訊平台


Folder Usage:

1.交易準則==>各式可能有用的資訊\
2.bargain==>籌碼追蹤程式\
  backtrack==>回測\
  determine==>判斷及資訊蒐集\
  monitor==>追蹤\
3.db_manipulate==>大量控制資料庫(如大批量刪除)\
4.shioaji==>永豐金證券報價api及程式\
5.stonk_bot==>用於line機器人的程式碼(備份，原檔@heroku)\
6.stonk_old==>歷代程式碼\
7.web==>網頁端程式擷取\
